E-Stop arrangements:

Setting up an E-Stop switch

An E-Stop must be built from a multi-part switch system. Each E-Stop should be comprised of two N-ON switches rated for more than 4A AC current, an indicator LED, and a low-voltage N-OFF switch.

Two E-Stops

This configuration is best for when you have an E-Stop on the controller box, and want another mounted closer to where the user will actually be working. A good example is having a second E-Stop right on the front of a Bridgeport Mill.

The control box's E-Stop [A] should be plugged in at all the green locations. The secondary E-Stop should be plugged in to all the blue locations.

One E-Stop

This configuration may be the most common. There is an E-Stop on the controller box, and none other.

The control box's E-Stop [A] should be plugged in at all the green locations. Jumpers will need placed where the blue 'U' shapes show connections for E-Stop B. Do not use the E-Stop-B JST plug.

No E-Stops

This configuration is only for the very safest of tools, when the control box is pperhaps undersized.

Jumpers will need placed where the blue 'U' shapes show connections for E-Stops A & B. Do not use the E-Stop-A or E-Stop-B JST plugs.

Current Sensor

The CT current sensor will be plugged into a 1/8" jack in this location. The current sensor itself should be of this kind of simple split-ring inductive sensor. The exact current rating of the sensor is not critical, but should be slightly above the value of the tool's circuit breaker, adjustments can be made in the firmware `offset` setting. [30A is a good fit for most.]

The current sensor should be clamped around the 'hot' line headed out to the tool, from the contactor.

I2C Card Reader

The RFID card reader connects using one of the two available I2C connections (it does not matter which of these).

The PN532 board should be like those shown here. They should be configured for I2C communication (switch DIP as needed) and have wires soldered to leav the flattest-possible surface to mount the board.

Bypass Key

The controller box will need a bypass key-switch, such as those shown here. This switch will need to conenct to a 2-pin JST conenctor as shown on the board.

Soldered-on Components

two components will need soldered by the end user.

The 5V transformer will need soldered into place in the location marked "PS1" on the board.

The Stage 2 relay will need soldered on where indicated. It must be an "Omron MY2-02" relay to fit, but can be rated for 110VAC or 220VAC.

It is advised, that after soldering on these components (and arranging the E-Stops) you should set the DIP pins, as appropriate.

Other Connections

There are a handful of other connections that must be made of the right side of the baord, to the mains-level control components that are panel-mounted on the box.

Incoming Power

The power lines must be connected as shown, with the correct polarity for any single-phase applications. 'H' for the Hot line. 'N' for the neutral line.

Out to Contactor

The external contactor should be mounted in the controller box, but the two wires coming out here should be used to activte it. These lines should only need to supply a relatively small current to control the state of the contactor.
